# ============================================================
# 配置.txt（記入例）… 写真の「置き場所」を書くファイル
# ------------------------------------------------------------
# 「記事.txt」に書いた文章の、どこに・どの写真を入れるかを書きます。
# 写真ファイルは images/news/ フォルダに置いておきます
# （ファイル名は半角英数字。日本語・空白は使わない）。
#
# 書き方（上から順番に読まれます）：
#   メイン画像：ファイル名｜写真の説明
#     → 記事の一番上に大きく表示する写真（無ければこの行を消す）
#   「見出し名」の後に画像：ファイル名｜説明
#     → 本文中のその見出しの直後に写真を入れる
#   画像グリッド：ファイル名, ファイル名, …｜説明1／説明2／…
#     → 複数枚を並べて表示（2〜4枚）
#   最後に画像：ファイル名｜説明
#     → 本文の一番最後に写真を入れる
#
# ※「｜」の右側（説明）は、目の不自由な方向けの代替テキスト
#   兼キャプションになります。短くて構いません。
# ============================================================
