# ★編集する人へ（メモ帳で編集できます）
#   ・1行に「年度：ウェブアプリのURL」を書きます。
#   ・新しい年度を追加するときは、下に1行足すだけでOKです。
#     （書く順番は自由。ページでは自動的に新しい年度が先頭になります）
#   ・URLがまだ無い年度は「2027：」のようにURL側を空にしておくと、
#     ページに「未設定」の案内が表示されます（エラーにはなりません）。
#   ・「#」で始まる行はメモです（表示に影響しません）。
#
# 【URLの取り方（年度ごとに1回だけ）】
#   1. その年度の「大会結果」スプレッドシートを開く
#   2. 拡張機能 → Apps Script → apps-script/results.gs の中身を貼り付け
#   3. デプロイ →「新しいデプロイ」→ 種類「ウェブアプリ」
#      → アクセスできるユーザー「全員」→ デプロイ
#   4. 表示された「ウェブアプリのURL」を下に貼り付ける
#
# 【大会の追加（ふだんの作業はこれだけ）】
#   ・その年度のスプレッドシートに新しいシート（タブ）を足すだけ。
#   ・タブ名：大会名〔半角スペース〕開催日｜開催場所
#     例）第2回トヨタ紡織記録挑戦会 2026年3月28日｜ウェーブスタジアム刈谷
#   ・ページでは開催日の新しい大会が自動で一番上に表示されます。
